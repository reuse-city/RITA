# Contributing to RITA

Thank you for considering contributing to RITA! This document provides guidelines for contributing to the project.

## Code of Conduct

This project follows a Code of Conduct that all contributors are expected to uphold. Please read [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).

## How to Contribute

1. **Find an Issue**
   - Check existing issues
   - Create new issues for bugs or features
   - Discuss changes in issues first

2. **Fork and Clone**
   - Fork the repository
   - Clone your fork locally
   - Set up development environment

3. **Create Changes**
   - Create a new branch
   - Write clear commit messages
   - Follow coding standards
   - Add tests and documentation

4. **Submit Changes**
   - Push to your fork
   - Create a Pull Request
   - Respond to feedback
